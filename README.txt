The program will automaticlly start once the "Run" button on the top left corner of the Processing IDE is clicked.

The program will first display all the segement read from two .in file. Then, A moving line 
will start moving from the left of the window to the right, the line will pause at each 
birth, death, and vertical event. If a horizontal segement is explored(i.e death event), it will be makred as green.
If a vertical is explored, it will be marked as red.

The user must press the key 'n' on the keyboard to make the program continue, all the intersection will be makred 
by a small ellpise.

Finally, this program will automaticlly create a file "intersection.out" in the same directory as the program.
